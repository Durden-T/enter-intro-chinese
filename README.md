# Enter.pro 产品使用手册 — 交互式演示

基于 Enter.pro 官方内容构建的交互式 Web 演示，无依赖纯原生 HTML/CSS/JS。

## 功能

- ⌨️ **键盘导航** — `←` `→` 切换，`Space` 自动播放
- 👆 **触屏支持** — 左右滑动切换
- ⊞ **幻灯片总览** — 按 `O` 或点击总览按钮
- ▶ **自动播放** — 4.5 秒/张，可随时暂停
- ⛶ **全屏模式** — 按 `F` 或点击全屏按钮
- 📊 **进度条** — 顶部红色进度指示

## 快捷键

| 键 | 功能 |
|---|---|
| `← →` | 上 / 下一张 |
| `Space` | 自动播放 / 暂停 |
| `O` | 幻灯片总览 |
| `F` | 全屏 |
| `Esc` | 关闭面板 |
| `?` | 显示快捷键帮助 |
| `Home / End` | 跳至第一 / 最后一张 |

## 文件结构

```

├── index.html   # 主文件，包含全部幻灯片内容
```

## 本地运行

直接用浏览器打开 `index.html` 即可，无需任何服务器或依赖。

## 导入 Enter.pro

1. 将本仓库推送到 GitHub
2. 在 Enter.pro 中选择 **Import from GitHub**
3. 选择本仓库，即可开始用 AI 迭代和扩展
